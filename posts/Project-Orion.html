Project Orion

High Concept: Third Person Space Shooter, in which the player will move and shoot enemies in a frenetic enviroment.

Instructor: Iker Silvano

Engine: 3D Game Engine from Scratch

Language: C++

Release: DigiPen Gallery

Overview: Project Orion is the third project I worked on at Digipen during the my Junior year. It took 9 months of development using a component based engine made from scratch in C++. The team was composed of 7 programmers.

Role: Engine Programmer

Participated in the creation of an C++ 3D data driven game engine from scratch by developing the serialization tool, Archetype system and event manager.

Designed and programmed a Serialization system for game objects on a component base methodology.

Developed a State preserving Particle System based on data oriented design and used to create different effects in the game

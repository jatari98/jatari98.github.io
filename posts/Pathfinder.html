Artificial Intelligence: A* Pathfinder

Course: CS380: Artificial Intelligence for Games

Instructor: Iker Silvano

Technology: C++ Framework using DirectX for rendering

Programming Language: C++

Overview:

The aim of this proyect is to implement A* algorithm used for pathfinding in a framework enviroment demo

This project implements the following features:

-Rubber banding and Smoothing post processing to improve the path the agent will follow.

-Variable weight for the cost of each node.

-Multiple heuristic computation functions: Euclidean, Octile, Chebyshev and Manhattan.
